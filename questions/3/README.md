# Question 3

![easy](https://img.shields.io/badge/-easy-green)

1000 未満の素数のうち、5番目に大きいものを出力してください。

<details>
<summary>ヒント - 素因数分解</summary>

`factor` コマンドで素因数分解できます。

```bash
$ factor <<< 123
123: 3 41

$ seq 10 | factor
1:
2: 2
3: 3
4: 2 2
5: 5
6: 2 3
7: 7
8: 2 2 2
9: 3 3
10: 2 5
```

</details>
